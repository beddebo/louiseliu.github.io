https://help.github.com/articles/generating-ssh-keys

2.0

常用命令：

-1. 下载安装git bash http://code.google.com/p/msysgit/

  1. 设置用户信息： git config --global user.email "xxx" git config --global user.name "xxx"

  2. 检出一个项目： git clone gitxxx

  3. 导入项目到eclipse 2.1 安装EGit

  4. 创建本地branch git branch newfeature

  5. 切换到新创建的branch git checkout newfeature

  6. 提交所做修改 5.1 获取修改的内容

     git status

    5.2 添加到commit

     git add file1 file2

    5.3 提交修改

     git commit -m "message"

  7. 回滚到指定版本 6.1 回滚到head

       git reset --hard HEAD

    6.2 回滚到指定版本

       通过git log查询各次提交的版本
   git reset --hard 98507c0010693fa1e122daae5ac32d593258c7ce

    6.3 回滚指定文件

       git checkout HEAD file/to/restore

  8. merge到本地master git checkout master git merge newfeature
